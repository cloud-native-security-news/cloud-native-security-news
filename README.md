# cloud-native-security-news

## 加入云原生安全资讯项目

1. 在 issues 中添加或处理 '待分析资讯'
2. 每天阅读安全咨询
3. 总结成文章, 模板参见 [security-research-specification](https://github.com/ssst0n3/security-research-specification/tree/main/cloud-native-security-news)
4. push 总结至 git 仓库
